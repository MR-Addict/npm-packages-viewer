# NPM Packages Viewer

> [!WARNING]
> 
> This repository has been deprecated, consider using the project [Packages Viewer](https://github.com/MR-Addict/packages-viewer)

## 1. What's it for?

Find npm packages latest stable version information.

## 2. How to development

Development:

```sh
npm run dev
```

Build:

```sh
npm run build
```

Preview:

```sh
npm run preview
```

Format:

```sh
npm run format
```
